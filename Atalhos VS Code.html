Atalhos que você pode usar no VS Code

CTRL

  • Abrir um novo arquivo.
  • Abrir um arquivo.
  • Pesquisar dentro do código.
  • Criar ou desfazer uma anotação.
  • Selecionar toda a linha.
  • Selecionar a próxima TAG idêntica (pode-se clicar "D" várias vezes).
  • Pesquisar uma linha.

SHIFT

  • Duplicar a linha.
  • Formatar o código de forma organizada

ALT

  • Mudar a linha de lugar.
  • Organizar um código quebrando as linhas para melhor organização.
